# ritzhouse-daily

リッツハウス全社員向けの朝礼・終礼プラグイン（Cowork Plugin Marketplace配布）。

## インストール

Coworkで以下を実行：

```
/plugin marketplace add ritzhouse/ritzhouse-daily
```

自動更新が有効なので、プラグインの更新は起動時に自動反映されます。

## スキル一覧

### おはよう（朝礼）
「おはよう」と話しかけると、朝礼ブリーフィングを配信します。
- 8:55のスケジューラが生成したブリーフィングをSlackから読み取って配信
- 運営ログから文脈を読み取り、前日の終礼・雑談の続きを踏まえた会話
- 個人向けNotionコーチング（脳カラム未入力、タスク少なすぎ、過負荷チェック）
- 会話の内容を運営ログに書き戻し

### しごおわ（終礼）
「しごおわ」「お疲れ」と話しかけると、終礼モードに入ります。
- 運営ログから今日の文脈を取得し、具体的な質問で聞き出し
- 今日の活動を会話形式で聞き出し
- ネクストアクションをNotionに登録（脳リソース必須）
- 終礼サマリーをSlackに投稿
- 全内容を運営ログに書き込み（翌朝のブリーフィング材料）

## アーキテクチャ：共有脳

```
[しごおわ] → 聞き出し → Notion登録 + Slack投稿 + ★運営ログ書込★
                                                    ↓
[スケジューラ 翌朝8:55] ← 運営ログを読む → ブリーフィング生成 → Slack投稿
                                                    ↓
[おはよう] ← Slackの投稿を読む + 運営ログから文脈 → ★運営ログ書込★
```

Notion「リッツハウス運営ログ」DBが全Coworkセッションの共有メモリとして機能します。各スキルが読み書きし、時間の経過とともにブリーフィングの質が向上します。

## 必要な接続

各社員のCoworkで以下のコネクタを接続してください：
- **Notion** — ALL TASKS, Projects, CRM, 運営ログへのアクセス
- **Slack** — #00-general への投稿権限
- **Google Calendar** — 予定の読み取り

## 朝礼スケジューラについて

8:55の自動ブリーフィング生成は安部のCoworkにScheduled Taskとして設定済みです（このプラグインには含まれません）。

## 開発

プラグインを更新するには：
1. このリポジトリのスキルファイルを編集
2. `plugin.json` の `version` をバンプ
3. mainブランチにプッシュ
4. 各社員のCowork起動時に自動更新される
